Welcome to Expert System Training

This is intended to be an expert system designed to train a machine using datasets from MedMNIST.

About Data

The dataset chosen for this expert system is one of MedMNIST's sets. The MedMNIST itself contains a total of twelve 2D image datasets and six 3D image datasets. For this system, the dataset DermaMNIST will be the focus set.

More about Dermanmnist

The modality for the DermaMNIST dataset is a Dermatoscope. There were a total of 10, 015 samples in this set. The aforementioned dermatoscope is a popularly used medical instrument amongst dermatologists and medical professionals alike. These instruments work by using light and magnification against the skin.
